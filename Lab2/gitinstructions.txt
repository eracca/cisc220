git pull origin [branch name] 

get the most up to date version of a branch from github

git branch

list branches, find out what branch you are in

git branch [branch name]

make new local branch with branch name, switch into that branch

git checkout [branch name] 

switch into branch name,  MAKE SURE YOU COMMIT CHANGES TO PREVIOUS BRANCH BEFORE DOING THIS

git add .
git commit -m "[commit message]"

commit all changes in branch to local branch

git push origin [branch name]

push local version of branch name to github, MAKE SURE YOU COMMIT BEFORE DOING THIS

git merge [branch name]

merge branch name with master, MAKE SURE YOU COMMIT, MAKE SURE YOU ARE IN MASTER
